廣東話聊天助手

呢個版本係純前端網站。用家自己輸入 Gemini API key，就可以直接喺瀏覽器用廣東話聊天。

How it works

API key 只會儲存在使用者自己部機個瀏覽器，同 Gemini 直接連線，唔需要你另外開後端。

Setup

連線設定

未設定 API key
取得 Gemini API key

如果部機係共用，唔好勾留 API key。語音固定使用 Gemini 3.1 Flash TTS Preview 生成。

Google 官方文件寫明 TTS model 係 text-only input、audio-only output，所以文字對話仍然需要另一個 chat model。香港廣東話語音仍然屬於 best-effort。
